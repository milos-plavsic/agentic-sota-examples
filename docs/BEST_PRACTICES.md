# Agent Engineering Best Practices (Applied)

- Graph-native orchestration with explicit state transitions (`langgraph`).
- Confidence-driven retry loops with explicit stop reasons.
- Contract-first output schema for reports, traces, and automation safety.
- Multi-axis evaluation (quality, calibration, latency, cost).
- Keyless PR CI for contributor reliability; secret-enabled nightly depth runs.
- Public internet data adapters with caching and deterministic fallbacks.
- Guardrail policies as declarative YAML with enforcement middleware.
- Human-review escalation artifacts for low-confidence/high-risk flows.
- Shared runtime primitives to minimize policy drift across projects.
